# Analyze Traffic with Cloudflare Radar

Help the user analyze internet traffic patterns, scan URLs for security issues, and understand internet trends using Cloudflare Radar.

## Capabilities

### Traffic Analysis
- Global internet traffic trends
- Regional traffic patterns
- Protocol distribution (HTTP, HTTPS, etc.)
- Traffic anomaly detection

### URL Scanning
- Scan URLs for malicious content
- Check phishing indicators
- Analyze page content and behavior
- Get security verdicts

### Security Intelligence
- Track attack trends
- Monitor DDoS activity
- Analyze bot traffic patterns
- Check domain reputation

## Workflow

1. **Understand the user's goal**
   - Traffic insights: regional patterns, trends, anomalies
   - Security analysis: URL scanning, threat detection
   - Research: internet trends, protocol adoption

2. **Use appropriate Radar tools**
   - For URL scanning: Use the URL scanner tools to analyze specific URLs
   - For traffic trends: Query traffic analytics data
   - For security insights: Check attack patterns and threat data

3. **Present findings clearly**
   - Summarize key findings
   - Highlight any security concerns
   - Provide actionable recommendations
   - Include relevant visualizations or data points

## Example Queries

- "What's the traffic pattern for my region?"
- "Scan this URL for security issues: example.com/suspicious"
- "What are the current DDoS attack trends?"
- "Show me HTTPS adoption rates globally"
